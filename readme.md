# SHIM

Tiny ARM Cortex-M0+ Development/Prototyping board.

<img src="docs/shim.png" width="40%" />

## Features

- USB VBUS, ext. VIN, ext. 5V, ext +3.3V 
- Push-buttons: User and Reset 
- LEDs: Power, 2 User LEDs
- Connectors: 2 GPIO, 4 PWM, 1 SWD
- Breadboard friendly
